Security Scorecard Report

What is the OpenSSF Scorecard? ➕

Scorecard checks for vulnerabilities of a given repo, including source code, build, dependencies, testing, and project maintenance. Please check documentation to learn more about the scoring criteria used

Get Started ➕

To analyze an open source project, please add the platform where the project is hosted, the organization and the repo name (e.g github.com/pytorch/pytorch) an click submit. You can try popular projects like Kubernetes, Pytorch or even the OpenSSF Scorecard!

How it works ➕

This app fetches data from the scorecard API (https://api.securityscorecards.dev/) and uses chart.js libraryto visualize the radar chart
